# rahulramath.github.io

My personal site. Plain HTML + CSS, no build step.

- `index.html` — main page
- `side-experiments/index.html` — casual project list
- `style.css` — the one stylesheet

## Adding a project

Open `side-experiments/index.html` and copy a `<p class="project">` block
under the right year label. Push to `main` and GitHub Pages redeploys
automatically.

## Local preview

Just open `index.html` in a browser, or:

```bash
python3 -m http.server 8000
```
